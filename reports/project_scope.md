
# Project Scope

## Project Title

Loan Default Prediction and Credit Risk Scorecard

## Objective

The objective of this project is to build a credit risk analytics and machine learning workflow that predicts borrower default probability, segments borrowers into risk bands, and provides business recommendations for underwriting and portfolio monitoring.

## In Scope

- Public loan dataset analysis
- Target variable creation
- Data cleaning and preprocessing
- SQL-based feature engineering and risk analysis
- Exploratory data analysis
- Baseline model development
- Logistic regression modeling
- Tree-based modeling
- Gradient boosting modeling
- Hyperparameter tuning
- Probability calibration
- Model evaluation using credit risk metrics
- Risk band creation
- Dashboard design
- Model card documentation
- Executive summary
- GitHub and LinkedIn project presentation

## Out of Scope

- Production deployment
- Automated credit approval or rejection
- Use of real customer data
- Regulatory model validation
- Direct integration with banking systems
- Credit bureau data integration
- Loan pricing optimization
- Collections strategy optimization
- Macroeconomic stress testing

## Stakeholders

- Credit Risk Analysts
- Risk Analytics Teams
- Underwriters
- Portfolio Managers
- Collections Teams
- Compliance and Model Risk Teams
- Executives

## Business Constraints

- Model output must be explainable to business users.
- Model should support decisions, not replace human judgment.
- False negatives may increase credit losses.
- False positives may reduce approval efficiency and customer experience.
- Thresholds must balance risk detection and manual review capacity.
- Model must be monitored for drift and performance degradation.

## Ethical Considerations

- Avoid using features that create unfair treatment.
- Document limitations and assumptions clearly.
- Do not use the model for automatic rejection without governance.
- Monitor model performance across borrower segments where possible.
- Protect borrower privacy and avoid exposing sensitive data.
